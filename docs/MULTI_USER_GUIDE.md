## 🔐 隔离规则

| 资源类型 | 隔离方式 | 说明 |
|----------|----------|------|
| **记忆** | 完全隔离 | 每个用户有自己的 `memory/` 和 `MEMORY.md` |
| **配置** | 完全隔离 | 每个用户有自己的 `user_config.json` |
| **技能数据** | 完全隔离 | 每个用户有自己的技能配置和数据 |
| **技能代码** | 共享 | 所有用户共享同一套技能代码 |
| **文档** | 共享 | 所有用户共享 `docs/` 目录 |

## 👥 多用户场景

当有**新的飞书用户**与我对话时：

1. **自动检测**：系统识别用户的 `sender_id`
2. **创建空间**：为新用户创建独立的 `users/{user_id}/` 目录
3. **完全隔离**：新用户的记忆、配置与**你的完全隔离**
4. **互不可见**：其他用户无法看到你的任何数据
